# Real-time Poll App

A Flutter application for creating and participating in real-time polls using Firebase Firestore.

## Features

*   Create new polls with multiple options.
*   Vote on existing polls.
*   View real-time updates of poll results.

## Getting Started

1.  Set up a Firebase project and enable Firestore.
2.  Add your Firebase configuration to the project.
3.  Run `flutter pub get` to install dependencies.
4.  Run the app on your device or emulator.
